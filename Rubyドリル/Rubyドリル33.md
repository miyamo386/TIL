# Rubyドリル３３（文字の順番を入れ替えて出力するプログラム実装）
### 自己解答
```ruby
def left2(str)
  puts "#{str[o..1]} #{str[2..5]}"
end

left2("Hello")
```

### 模範解答
```ruby
def left2(str)
  puts str[2..-1] + str[0..1]  
end

# 呼び出し例
left2("Hello")
```

### 学んだこと
- 文字列に対して範囲指定する方法は、[]を用いて取得したい文字列の番号で範囲指定して文字列を取得します。
- 文字列を繋げる際、＋を用います。
