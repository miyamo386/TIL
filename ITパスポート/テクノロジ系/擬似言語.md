# ⒈ 擬似言語
- 試験用に定義された言語。
- 記述形式は試験中も確認できるので暗記しなくてもいい。

# ⒉ 変数と型
## ❶ 変数
- プログラムで扱うデータに名前を付けた、データを入れる箱のようなもの。

## ❷ 変数の宣言
- 変数を作成すること。
- 擬似言語では「型名：変数名」
- 変数宣言と代入を一度に行える「整数型：num ← １０」
- 複数の変数を一度に宣言できる。「整数型：x，y」

## ❸ 代入
- 変数にデータを設定すること。
- 擬似言語では「変数名 ← 値」

## ❹ 型
- 変数に代入できる値の種類のこと。
- 文字列型のみ、変数に代入する値を””で囲みます。

| 型 | 概要 | 例 |
| --- | --- | --- |
| 整数型 | 整数 | １，１０，−１０ |
| 実数型 | 実数（小数を含む） | １.５，−１０.５ |
| 文字列型 | 文字 | 学研太郎，ひとつ花子 |
| 論理型 | 真偽値 | ture，falseのみ |

## ❺ 代入でよく使われる書き方
- 変数の値に、計算をした結果を代入し直す書き方で、例えば、「num ← num + 10」の場合，変数の値に10を加算した結果を，変数に代入するという意味になります。

# ⒊ 選択〜if文

## ❶ if
- ifは「もしも〜」という英単語で、選択構造を表す書き方。

```python
if(age < 18)
 "未成年です"を出力する
endif
```

## ❷ else（エルス）
- elseは「それ以外」という英単語で、ifでは条件を満たさない場合の処理を表す書き方。
- 1つのif文では、1つしか記述できない。

```python
if(age < 18)
 "未成年です"を出力する
else
 "成人です"を出力する
endif
```

## ❸ elseif（エルスイフ）
- ifを使って条件を複数使用したい場合の処理を表す書き方。

```python
if(age < 18)
 "未成年です"を出力する
elseif(age < 65)
 "成人です"を出力する
else
 "高齢者です"を出力する
endif
```

# ⒋ 繰り返し
## ❶ for文
- forは「〜している間」という意味の英単語で、処理の繰り返しを表す書き方。
- 例えば、「変数numを6から10まで1ずつ増やしている間は，変数numの値を表示する」
    ### ◆ 初期値
    - 繰り返し処理の1回目の値のことで、上記では「6」になります。
    
    ### ◆ 増分値
    - 繰り返し処理で、1回で増える値のことで、上記では「1」になります。
    
    ### ◆ 終値
    - 繰り返し処理が終了した時点の値のことで、上記では「10」になります。
        
        ```python
        整数型: num
        for(numを6から10まで1ずつ増やす)
          numを出力する
        endfor
        ```
        
        | 繰り返しの回数 | １ | ２ | ３ | ４ | ５ |
        | --- | --- | --- | --- | --- | --- |
        | 変数numの値 | 6 | ７ | 8 | 9 | 1０ |

## ❷ while文
- 繰り返す回数が決まっていない場合に用いられる書き方。
    
    ```python
    整数型: num
     while(num < 5)
      numを出力する
      num ← num + 1
    endwhile
    ```
## ❸ do~while文
- 処理を実行してから条件の判定を行う繰り返し処理です。

# ⒌ 関数
- プログラムの中で、特定の処理をする小さなプログラムのグループ（集合体）みたいなもの。
- 処理を役割ごとに分けられるので、プログラムが読みやくすなる，プログラムの修正がやりやすいなどのメリットがあります。

## ❶ 関数の宣言
- 関数を作成すること。
- 擬似言語では（〇〇関数名）と書きます。

## ❷ 関数の呼び出し
- 作成した関数を使用する時の書き方。

## ❸ 引数
- 関数に渡す値のこと。

## ❹ 戻り値
- 関数から、呼び出し元に戻った値のこと。
- 擬似言語では、「⚪︎戻り値の型名：関数名（引数の型名：引数名）」

# ⒍ 配列
- 1つの変数に、同じ種類のデータを順番に並べて複数のデータを扱う仕組みのこと。
- 擬似言語では、「配列の型名：変数名」と記述し，代入する値をカンマで区切り、{}で囲みます。

## ❶ 要素
- 配列に入った1つ1つの値（箱）のこと。

## ❷ 要素数
- 要素の数のこと。

## ❸ 要素番号
- 要素1つ1つに割り当てらた番号のこと。
- データを取り出したり，データに再代入する時に、要素番号を指定します。
